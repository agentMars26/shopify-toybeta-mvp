# Shopify Theme MVP - Toybeta Inspired

Goal: build a Shopify theme MVP inspired by toybeta.com for selling to ecommerce factories as a high-conversion demo.

Scope for MVP:
- Home page
- Collection page
- Product page
- Announcement / promo bars
- Featured collections / hot selling sections
- Testimonials / reviews placeholder
- Email subscribe section
- Mobile-friendly responsive layout

Notes:
- Target stack: Shopify theme architecture
- Prioritize reusable sections/snippets
- Keep theme clean enough to later connect to a real Shopify store
